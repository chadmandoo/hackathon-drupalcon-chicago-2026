# Judging rubric

## Eligibility gate

An entry is eligible only if it includes:
1. a real agent-work moment
2. at least one AX artifact
3. README
4. Agent Run Log
5. Agent Experience Report
6. an identified team lead who is attending DrupalCon Chicago in person and can represent the submission on Wednesday afternoon

## Recommended scoring weights

| Criterion | Weight | What good looks like |
|---|---:|---|
| Agent Success | 30 | The agent completed useful work or made credible progress toward a real goal |
| AX Quality | 25 | The submission improves guidance, validation, or future first-pass success |
| Drupal-in-the-loop | 20 | Drupal acts as governor through workflow, permissions, schema, diffs, or auditability |
| Openness | 15 | Works with more than one tool or follows open patterns others can reuse |
| Impact | 10 | Improves real build, ops, or contribution workflows |

## Scoring guide

### 5 — Exceptional
Clear, reusable, and demonstrably valuable. Strong evidence. Minimal ambiguity.

### 4 — Strong
Works well, mostly complete, and useful beyond a one-off demo.

### 3 — Solid
Meets requirements and shows promise, but needs polish or stronger evidence.

### 2 — Partial
Interesting direction, but important parts are missing or weakly demonstrated.

### 1 — Thin
Does not yet show meaningful agent success or durable AX improvement.

## Category winners

### Best Overall
Highest total score across the full rubric.

### Best Agent
The most impressive, funny, or memorable use of an agent — while still producing real work.

### Best Contribution Back to Drupal
The strongest community contribution, such as:
- patch
- PR
- issue + fix
- reusable improvement to Drupal AI / AX foundations

Entries in this category should be capable of being contributed back under Drupal-compatible licensing. For Drupal-derivative code intended for Drupal.org-hosted projects, that generally means code that can be released under **GPL-2.0-or-later**.

## Tie-breakers

Use these in order:
1. stronger Drupal-in-the-loop evidence
2. more reusable AX artifact
3. clearer Agent Experience Report
4. better contribution back to Drupal

## Recommended judging flow

1. Run eligibility check
2. Independent human scoring pass
3. AI-assisted summary/comparison pass
4. Human discussion and final decision
5. Write 2–3 sentence winner blurbs

## Judging note for public copy

Suggested wording:

> Entries will be reviewed by human judges, with AI-assisted scoring used to help summarize patterns and compare submissions. Final decisions are made by humans.
